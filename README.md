# ODA Web Components

Repository này chứa tất cả các UI components được collect từ các repositories:
- `oda-web`
- `oda-web-admin`

## Cấu trúc thư mục

```
oda-web-component/
├── oda-web/
│   └── components/
│       ├── shared-components/    # Các components dùng chung
│       ├── styled-components/    # Các styled components
│       ├── common/               # Components thông thường
│       ├── icon/                 # Icon components
│       └── app/                  # App-specific components
│
└── oda-web-admin/
    └── components/
        ├── shared-components/    # Các components dùng chung
        ├── styled-components/    # Các styled components
        ├── common/               # Components thông thường
        ├── icon/                 # Icon components
        └── PageHeader/           # Page header components
```
